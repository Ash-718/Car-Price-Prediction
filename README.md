# 🚗 Car Price Prediction

Machine Learning project that predicts the selling price of used cars using historical vehicle data from Quikr.

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training using Scikit-learn
- Price Prediction
- Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Dataset
Quikr Used Cars Dataset
